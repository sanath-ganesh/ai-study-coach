AI Study Coach

A RAG-based generative AI tutor for Introduction to Data Structures.

View on GitHub Watch Demo Video

What It Does

  • Answers questions about your course notes using Retrieval-Augmented Generation (RAG).
  • Generates practice quizzes from your materials (synthetic data generation).
  • Explains topics simply or with analogies using prompt-engineered tutor modes.

How It Works

  1. Upload PDFs of your course materials into data/raw/.
  2. Run the ingestion script to build a vector index over your notes.
  3. Interact through the Streamlit web app (question answering, explanations, quiz generation).

Under the Hood

  • FastAPI backend providing /ask and /generate_synthetic endpoints.
  • OpenAI models for embeddings and chat completion.
  • ChromaDB as a persistent vector database.
  • Prompt engineering for different tutor modes and quiz generation.
